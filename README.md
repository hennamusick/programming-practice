# Programming Practice

A curated collection of programming questions and solutions organized by language and topic.

## Languages

| Language | Folder |
|---|---|
| Java | [`java/`](./java/) |
| Python | [`python/`](./python/) |
| TypeScript | [`typescript/`](./typescript/) |
| JavaScript | [`javascript/`](./javascript/) |

## Topics Covered

Each language folder is organized into the following topic categories:

| Topic | Description |
|---|---|
| `arrays` | Array manipulation, sliding window, two pointers |
| `strings` | String parsing, pattern matching, anagrams |
| `linked-lists` | Singly/doubly linked lists, reversal, cycle detection |
| `trees` | Binary trees, BSTs, traversals, lowest common ancestor |
| `graphs` | BFS, DFS, topological sort, shortest path |
| `dynamic-programming` | Memoization, tabulation, classic DP problems |
| `sorting-searching` | Sorting algorithms, binary search |
| `recursion` | Backtracking, divide and conquer |
| `hash-maps` | Frequency counts, grouping, caching |
| `stacks-queues` | Monotonic stacks, queue-based BFS, deque |
| `math` | Number theory, combinatorics, bit manipulation |

## How to Contribute

1. Pick a language folder (e.g., `java/`).
2. Navigate to the relevant topic folder (e.g., `java/arrays/`).
3. Add a new file named after the problem (e.g., `TwoSum.java`).
4. Include a comment block at the top of the file with:
   - Problem description
   - Constraints
   - Example input/output
   - Time and space complexity

## Naming Conventions

- **Java**: `PascalCase.java` (e.g., `TwoSum.java`)
- **Python**: `snake_case.py` (e.g., `two_sum.py`)
- **TypeScript**: `camelCase.ts` (e.g., `twoSum.ts`)
- **JavaScript**: `camelCase.js` (e.g., `twoSum.js`)
