# Java

Solutions to programming problems written in Java.

## Prerequisites

- Java 17+ (LTS recommended)
- A build tool such as [Maven](https://maven.apache.org/) or [Gradle](https://gradle.org/) (optional)

## Running a Solution

Compile and run any file directly with `javac` / `java`:

```bash
javac arrays/TwoSum.java
java arrays.TwoSum
```
