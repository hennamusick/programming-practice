# TypeScript

Solutions to programming problems written in TypeScript.

## Prerequisites

- [Node.js](https://nodejs.org/) 18+
- [TypeScript](https://www.typescriptlang.org/) (`npm install -g typescript`)

## Running a Solution

```bash
# Compile then run
tsc arrays/twoSum.ts && node arrays/twoSum.js

# Or use ts-node for a single step
npx ts-node arrays/twoSum.ts
```
